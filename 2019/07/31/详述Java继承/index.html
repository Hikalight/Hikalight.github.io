详述Java继承

发表于 2019-07-31 | 分类于 Java学习之道 | | 阅读次数:
字数统计: 325

同C++一样，Java也有继承特性，这是面向对象编程语音的普遍特征。不同类之间的继承，可以提高代码的重用性，本文侧重于叙述Java和C++在继承方面的不同之处

Java继承格式

Java继承特性

  • 子类继承父类的属性和方法，private不可继承
  • 如果一个类没有显式继承某个类，则该类默认继承自Object类
  • Java继承为单继承，一个子类只能继承一个父类
  • final 不能有子类Void、String、Class、Scanner、System
    -

与C++不同，Java并不支持多继承，因为会导致Diamond问题

super关键字

Java只能用super调用父类，不能调用爷类

implements关键字

当类继承接口时，可以使用implements关键字，让Java能够多继承。

final关键字

  • final类不能被继承，没有子类，final类中的方法默认是final的。
  • final方法不能被子类的方法覆盖，但可以被继承。
  • final成员变量表示常量，只能被赋值一次，赋值后值不再改变。
  • final不能用于修饰构造方法。
