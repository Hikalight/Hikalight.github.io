基本类型详述

  1. 整型（short byte int long)
    用于声明整数变量，对于这四个整型数据类型，系统分配给其的空间依次增大。通常使用int，long，而short和byte这两种类型，由于现代计算机的性能足够强，故基本不再使用。
  2. 浮点型（float,double）
    用于声明带小数点的变量，精度要比整型高。对于这两种数据类型，系统分配给其空间也依次增大。且有一固定空间用于存放小数点。

  3. 字符型（char）

  4. 布尔型（boolean）
    布尔型只有true和false两种值，且默认为false，通常用作flag。

数据类型强制转换

与c语言和c++相同，Java也可以进行强制数据类型转换，简单说一下，Java的显式转换如double num=123.123; (int)num;,此时会精度缺失，num的值变成123。而隐式转换只能由表示范围大的数据类型转化为表示范围小的数据类型。
